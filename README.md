# eDistrict UP Status Checker

eDistrict Uttar Pradesh के applications का status चेक करने के लिए web tool.

## Features
- ✅ Simple और clean interface
- ✅ Real-time status checking
- ✅ Captcha support (manual)
- ✅ API endpoints available

## कैसे use करें?

### Local Development
```bash
# Install dependencies
